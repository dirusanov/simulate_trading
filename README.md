## О программе

Программа представляет собой симулятор торговли на бирже. Функция `generate_movement` возвращает 1 (цена идёт вверх) или -1 (цена идёт вниз). Реализация старгии сделана максимально простой для лучшего понимания.
```
from random import random


def generate_movement():
    movement = -1 if random() < 0.5 else 1
    return movement
```

Вероятность выпадания каждого из двух чисел (1 и -1), условно можно принять за 50%. Это как при подбрасывании монетки, вероятность выпадания орла 50% и решки 50%. 
Суть стратегии: орёл выигрываем $10, решка проигрываем $10.
За каждую торговую сессию полученные числа суммируются, это и представляет собой потенциальный заработок.
Например 3 раза выпала `1` и 2 раза `-1`, получаем `1 + 1 + 1 - 1 - 1 = 1`.

*Баланс может быть разным, но на длинной дистанции он будет стремиться к нулю.*

Основные цели:
* Имитировать работу торговой стратегии
* Реализовать взаимодействие приложения на React JS с WebSocket
* Создать WebSocket используя фреймворк FastApi и попробовать новую структуру приложения

Есть выпадающий список тикеров, именна в формате `ticker__<number>`. 
`<number>` это коэфициент на который умножается возвращаемое число функции `generate_movement`. Для большего интереса, если вы хотите торговать по крупному :)

## Для начала

Должен быть установлен Docker и Docker Compose.

### Установка

1. Склонировать репозиторий
   ```sh
   git clone https://github.com/dirusanov/simulate_trading.git
   ```
2. Запустить докер контейнеры
   ```sh
   docker-compose up
   ```
3. Открыть в браузере
   ```sh
   http://0.0.0.0:3000
   ```
4. Смотреть как изменяется баланс :)
